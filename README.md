# Arkanoid Game

This is a simple Arkanoid game built using HTML canvas and JavaScript.

## How to Play

- Use the left and right arrow keys to move the paddle.
- Hit the ball with the paddle to bounce it and break the bricks.
- Break all the bricks to win the game.
- If the ball falls below the paddle, the game ends.

## Features

- Score tracking: Each brick broken adds to the score.
- Reset button: Restart the game at any time.
- Win condition: Display a message when all bricks are broken.

## Files

- `index.html`: Contains the HTML structure of the game.
- `style.css`: Contains the CSS styles for the game layout.
- `script.js`: Contains the JavaScript code for the game logic.

## How to Run

1. Clone the repository to your local machine.
2. Open `index.html` in a web browser.
